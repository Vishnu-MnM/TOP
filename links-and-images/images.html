Images

Let's take a look at the most common image formats. Go back to links for more or extras if you want to see some additional info.

JPG

JPG is a good format for photos.

GIF

GIFs are good for small animations.

PNG

PNGs are good for icons, technical diagrams, logos, etc.

SVG

SVGs are awesome, use them wherever you can. But to make them render correctly across browsers, some additional work is required. If the SVGs contain a lot of text, then will impact the size of the website, so use them with caution.
